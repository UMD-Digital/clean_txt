# Clean txt shell script

## Usage:

```bash
./clean_txt.sh -i /path/to/input-file.txt -o /path/to/output-file.txt
```

## Options

| Flags | Required? | Description              |
| ----- | :-------: | ------------------------ |
| -i    |    ✅     | Path to the input file.  |
| -o    |    ✅     | Path to the output file. |
